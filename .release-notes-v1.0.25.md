## DLBPlus 1.0.25

### What's new
- App Update dialog: removed Ignore — actions are now Close + Download / Downloading… / Install only
- App Update dialog: narrower action buttons with balanced two-button layout
- App Update dialog: Downloading state keeps full dark-blue styling (no faded disabled look)
- Preserves changelog scroll, DPAD escape to buttons, and white focus rings

### Install / update
- In-place update from **v1.0.0** through **v1.0.24** is supported (same `dlbplus.jks` signing key).
- Primary asset: `app-universal-release.apk`
