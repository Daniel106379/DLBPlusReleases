## DLBPlus 1.0.74

### What's new
- **Show all in rail:** MENU / long-press on Home catalog titles adds "Show all in [rail name]"
- Opens a portrait poster grid for that rail
- Loads ~100 items at a time and prefetches the next page near ~70

### Install / update
- In-place update from prior dlbplus.jks builds is supported.
- Primary asset: app-universal-release.apk
- Fire TV (armeabi-v7a): app-armeabi-v7a-release.apk
