## DLBPlus 1.0.16

### What's new
- Fixed main-profile login crash caused by full-series episode loading on Continue Watching
- Restored Continue Watching and Next Up for shows with progress in later seasons
- Scoped TMDB episode loading to watched seasons only (prevents OOM on long-running series)
- Hardened watch-progress reads and CW enrichment during remote sync

### Install / update
- In-place update from **v1.0.0** through **v1.0.15** is supported (same `dlbplus.jks` signing key).
- Upload `app-universal-release.apk` as the primary asset.
