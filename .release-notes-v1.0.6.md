## DLBPlus 1.0.6

### What's new
- Proper version bump so the in-app updater can install on devices on v1.0.4 or v1.0.5 (higher versionCode required by Android)
- Includes sidebar/nav colour fix from v1.0.5 (solid nav background, no grey scrim/tint)
- Continued `dlbplus.jks` signing era

### Install / update
- In-place update from **v1.0.0** through **v1.0.5** is supported (same signing key).
- Upload `app-universal-release.apk` as the primary asset.
