## DLBPlus 1.0.26

### What's new
- Backend catalog home rail parity: trending, top20, new, and popular movie/TV rails from DLBPlus API
- Silent in-rail paging for backend catalog rails (loads next page near the end without resetting focus)
- Home hides legacy backend rails (Recently Synced, trailer rails, provider rails on Home)
- Backend Top Movies / Top TV restored to portrait numbered ranking cards
- Coming Soon still sourced from live MdbList (not backend)

### Install / update
- In-place update from **v1.0.0** through **v1.0.25** is supported (same `dlbplus.jks` signing key).
- Primary asset: `app-universal-release.apk`
