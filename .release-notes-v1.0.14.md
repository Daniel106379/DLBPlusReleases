## DLBPlus 1.0.14

### What's new
- Removed global landscape/portrait override from Layout Settings
- Fixed Top ranking rails to stay portrait with giant Netflix-style rank numbers
- Kept normal movie/TV, Continue Watching, and Coming Soon rails landscape
- Preserved card width and corner radius layout settings
- Restored expand poster controls in Layout Settings for Modern view

### Install / update
- In-place update from **v1.0.0** through **v1.0.13** is supported (same `dlbplus.jks` signing key).
- Upload `app-universal-release.apk` as the primary asset.
