## DLBPlus 1.0.28

### What's new
- **Continue Watching hero:** metadata and synopsis sync immediately when enrichment completes — no need to move focus away and back
- **DLBPlus Backend catalog:** landscape card artwork is now authoritative; post-paint TMDb enrichment no longer repaints backend rails
- Backend mode home rails render once with stable logos and title-art overlays

### Includes from 1.0.27
- Simplified Catalog Source settings (TMDb/Discover+ and DLBPlus Backend only)
- Backend catalog home rail parity and silent in-rail paging
- Portrait numbered top20 cards; legacy/provider rails hidden from Home

### Install / update
- In-place update from **v1.0.0** through **v1.0.27** is supported (same `dlbplus.jks` signing key).
- Primary asset: `app-universal-release.apk`
