## DLBPlus 1.0.13

### What's new
- Improved Netflix-style ranking rails (Top Movies / Top TV Shows This Week)
- Fixed modal focus shadow rendering on update dialog buttons
- Improved Top 10 visual styling: larger bold matte rank numerals, tighter spacing, subtle focus animation

### Install / update
- In-place update from **v1.0.0** through **v1.0.12** is supported (same `dlbplus.jks` signing key).
- Upload `app-universal-release.apk` as the primary asset.
