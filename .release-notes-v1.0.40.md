## DLBPlus 1.0.40

### What's new
- **Continue Watching Home return focus:** Fixed focus drift when returning to Home — item 0 stays focused with correct scroll and hero
- **Stable CW LazyRow keys:** Item identity no longer changes after TMDb episode metadata enrichment
- **Next Up ordering:** Next Up no longer re-sorts in-progress Continue Watching entries unexpectedly
- **Home entry CW lock:** Focus, horizontal scroll, and hero stay locked to the entry item by stable key until you move away
- **My List hero logo:** Title logo restored on My List hero; landscape cards remain logo-free

### Install / update
- In-place update from **v1.0.0** through **v1.0.39** is supported (same `dlbplus.jks` signing key).
- Primary asset: `app-universal-release.apk`
