## DLBPlus 1.0.17

### What's new
- TMDb landscape cards prefer English title/text backdrops when available (w1280)
- Hero backgrounds stay on normal cinematic backdrops only (w1920), never English title art
- Logo overlay suppressed when the card image already contains embedded title artwork
- Applied consistently across home, search, My List, collections, and recommendation rails
- Forward-compatible backend DTO mapping for future embedded-title fields

### Install / update
- In-place update from **v1.0.0** through **v1.0.16** is supported (same `dlbplus.jks` signing key).
- Upload `app-universal-release.apk` as the primary asset.
