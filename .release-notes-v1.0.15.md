## DLBPlus 1.0.15

### What's new
- Search page now opens with browse rails instead of keyboard
- Added TMDb Trending, Collections, Franchises and extra browse rails
- Added collapsible TV keyboard with letters/numbers toggle
- Added collection/franchise result pages
- Improved DPAD focus behaviour
- Sorted collection/franchise items by release date
- Cleaned search hero metadata without affecting Home hero

### Install / update
- In-place update from **v1.0.0** through **v1.0.14** is supported (same `dlbplus.jks` signing key).
- Upload `app-universal-release.apk` as the primary asset.
