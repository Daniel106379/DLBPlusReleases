## DLBPlus 1.0.4

### What's new
- Match ARVIO-style hero provider logos using bundled SVG wordmarks (not TMDb square tiles)
- Compact inline provider logo (16×52 dp) on hero row 2 after release status
- Hero row 1: genre, runtime, year, IMDb, and UK age rating only
- UK flatrate providers only: Netflix, Prime Video, Disney+, Apple TV+, Paramount+, Max
- One provider per title, no duplicates, no UK broadcasters on hero

### Install / update
- In-place update from **v1.0.0** through **v1.0.2** is supported (same signing key).
- Upload `app-universal-release.apk` as the primary asset.
