## DLBPlus 1.0.36

### What's new
- **Autoupdater verification release:** Same build as 1.0.35, published so in-app update can be tested from 1.0.35 (build 36) → 1.0.36 (build 37).

### Included since 1.0.35
- **Backend Movies/TV genre rails:** Movies and TV hubs load trending/new/popular and genre rows from the DLBPlus backend catalog
- **Backend-only Continue Watching:** Home CW and Next Up show only backend-resolvable `tmdb:` titles
- **Backend-only My List:** Local My List hydrates via backend detail; non-backend rows are hidden
- **Backend-authoritative Details:** Detail pages skip live TMDb non-episode enrichment in backend mode
- **Backend Search:** Search and browse rails use backend movies/series/list endpoints
- **Top 20 portrait posters:** Ranked Home rails use backend `portrait_poster_url`
