## DLBPlus 1.0.68

### What's new
- **Delayed rail reset / vertical judder:** Departed catalog rails now reset to item 0 immediately when focus leaves the row (no delayed snap-back)
- Vertical row changes use a single scroll owner to reduce judder

### Install / update
- In-place update from prior dlbplus.jks builds is supported.
- Primary asset: app-universal-release.apk
- Fire TV (armeabi-v7a): app-armeabi-v7a-release.apk
