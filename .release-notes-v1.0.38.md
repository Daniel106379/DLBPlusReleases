## DLBPlus 1.0.38

### What's new
- **Search Play/Pause submit:** Play/Pause (and related media keys) submit the current query from anywhere on the Search screen, including the on-screen keyboard
- **Continue Watching movie artwork:** Movie CW cards use the same landscape card resolver as Home/Trending/Search rails via hydrated catalog meta preview
- **My List landscape cards:** Removed title-logo overlay on landscape cards; hero logos unchanged
- **Hero backdrop fallback:** Backend hero mapping skips title-card artwork when it was stored as the hero URL (fixes Game of Thrones grey edge artefact)
- **Episodes DPAD (long seasons):** Season row scrolls to selected season before focus; episode cards navigate up to season chips; no focus trap looping back to tabs
- **Continue Watching scroll reset:** Returning to Home resets CW focus index and horizontal scroll to item 0

### Install / update
- In-place update from **v1.0.0** through **v1.0.37** is supported (same `dlbplus.jks` signing key).
- Primary asset: `app-universal-release.apk`
