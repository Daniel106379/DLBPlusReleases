## DLBPlus 1.0.35

### What's new
- **Backend Movies/TV genre rails:** Movies and TV hubs load trending/new/popular and genre rows from the DLBPlus backend catalog (no live Discover+ genre rails in backend mode)
- **Backend-only Continue Watching:** Home CW and Next Up show only backend-resolvable `tmdb:` titles; orphaned progress rows are hidden without deleting stored progress
- **Backend-only My List:** Local My List hydrates via backend detail; non-backend rows are hidden without deleting stored library records
- **Backend-authoritative Details:** Detail pages skip live TMDb non-episode enrichment when Catalog Source = DLBPlus Backend
- **Backend Search:** Search and browse rails use backend movies/series/list endpoints in backend mode
- **Top 20 portrait posters:** Ranked Home rails use backend `portrait_poster_url` for Netflix-style ranking cards
