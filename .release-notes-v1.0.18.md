## DLBPlus 1.0.18

### What's new
- Netflix-style Top 10 ranked portrait cards (Top Movies / Top TV Shows This Week)
- Large white rank numerals attached to each poster, with proper two-digit layout for ranks 10–20
- Compact card sizing tuned for ~5–6 items visible on 1080p TV
- Includes v1.0.17 TMDb English title backdrop improvements for landscape cards

### Install / update
- In-place update from **v1.0.0** through **v1.0.17** is supported (same `dlbplus.jks` signing key).
- Upload `app-universal-release.apk` as the primary asset.
