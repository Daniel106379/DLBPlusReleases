## DLBPlus 1.0.37

### What's new
- **Backend search Cast row support:** Unified `/api/dlb/tmdb/search` grouped rows (Movies, TV Shows, Cast) with backend person detail/filmography
- **Play/Pause submits search:** Fire TV remote Play/Pause immediately submits the current search query
- **Continue Watching artwork consistency:** Movie CW cards use the same landscape card artwork resolver as Home/Search rails
- **My List metadata/artwork consistency:** Removed redundant Movie/TV Show hero labels; cleaner genres; backend-authoritative hydration
- **Details info focus restore:** Closing the synopsis overlay returns focus to the info button
- **Episodes DPAD navigation fix:** Season chips always link down into episodes on long-running shows
- **Chip colour state fix:** Season chips render correct selected colour on first paint
- **Resume button progress text:** Resume shows elapsed time (e.g. `Resume (24:33)`, `Resume S18E2 (12:17)`) when progress is available
- **Hero backdrop rendering:** Pure-black backdrop base and right-edge scrim reduce grey/beige edge artefacts

### Install / update
- In-place update from **v1.0.0** through **v1.0.36** is supported (same `dlbplus.jks` signing key).
- Primary asset: `app-universal-release.apk`
